🎬 **Movie App – React Native**

A modern movie browsing app built with React Native that allows users to explore trending and top-rated movies, view detailed information including ratings, release dates, and cast, search movies globally, and discover actor profiles, all powered by the TMDB API with a clean and optimized UI for both Android and iOS using technologies like React Navigation, NativeWind, Axios, and React Hooks.
